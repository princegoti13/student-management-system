# 🎓 Student Management System

A Complete Student Management System Developed Using **PHP, MySQL, Bootstrap 5, HTML, CSS, JavaScript**.

This project allows administrators to manage students, attendance, profiles, and passwords while providing students with their own secure dashboard.

# 🚀 Installation

1. Clone Repository

```
git clone https://github.com/your-username/student-management-system.git
```

2. Open Project

3. Import Database

4. Configure Database Connection

```
db.php
```

5. Run Docker

```
docker-compose up --build
```

6. Open Browser

```
http://localhost:9090
```
